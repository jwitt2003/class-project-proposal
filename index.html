Application Overview

Escape the Maze is a 2D puzzle game where players navigate through mazes filled with obstacles, dead-ends, and a few hidden power-ups. The goal is to find the quickest route to the exit before the timer runs out. The game is easy to pick up and focuses on teaching problem-solving skills, making it ideal for new players and casual gamers.

End User Perspective: Players enjoy a simple, intuitive game interface with straightforward controls, making it accessible for anyone. The mazes become progressively harder, providing an increasing level of challenge that tests both patience and logic.

Feature List

  • Basic Character Movement: Use arrow keys or WASD for movement.
  • Timer: A countdown clock that requires players to find the exit before time runs out.
  • Maze Levels: A series of progressively challenging mazes with various layouts.
  • Collectibles: Small power-ups scattered across the maze to give players extra time or a speed boost.
  • Score Display: Visual feedback on time remaining and levels completed.

Future Feature List

  • Multiplayer Mode: Add a two-player mode for a competitive challenge.
  • Custom Mazes: Players can create their own mazes for friends to solve.
  • Randomized Mazes: A mode with randomly generated mazes for endless replayability.
  • Hints System: Provide hints for players stuck in more difficult mazes.
  • Mobile Compatibility: Make the game mobile-friendly to reach a larger audience.

Basic Technical Features

  • Responsive Layout: The game adapts to various screen sizes (desktop, tablet).
  • Keyboard Input Handling: Capture and process keyboard input to control character movement.
  • Collision Detection: Ensures the player can't walk through maze walls or obstacles.
  • Score and Timer System: Keeps track of the player’s score and remaining time.
  • Local Storage for High Scores: Save top scores on the player’s device for replay motivation.

Advanced Technical Features (Optional)

  • Maze Generation Algorithm: Randomly generate maze layouts to enhance replayability.
